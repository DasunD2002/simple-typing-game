Typing Game 

This is a simple Typing Game made using HTML, CSS, and JavaScript.
The goal of the game is to type the correct letter before the time runs out.

How to Play

A random capital letter appears in the circle.
Press the same key on your keyboard.
If the key is correct → Correct count increases.
If the key is wrong → Wrong count increases.
If time runs out → Missed count increases.

Controls

Keyboard – Type the shown letter
Stop Button – Pause the game
Re-Start Button – Start the game again
Speed Slider – Change how fast the letters change

Game Features

Random letter generator (A–Z)
Progress bar to show time

Counts for:

Correct
Wrong
Missed

Adjustable speed
Colorful and animated UI

Technologies Used

HTML
CSS
JavaScript

How to Run

Download or copy the files.
Open the HTML file in any web browser.
Start typing and enjoy the game!
